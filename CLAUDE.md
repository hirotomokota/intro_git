# CLAUDE.md

このファイルは、AIアシスタントがこのリポジトリを扱う際のガイドラインを提供します。

## プロジェクト概要

Git の基本操作を学習・練習するためのリポジトリです。作成者が Git のワークフローを段階的に習得する過程を記録しています。

## リポジトリ構成

```
intro_git/
├── CLAUDE.md          # AIアシスタント向けガイドライン（本ファイル）
├── first.txt.txt      # Git学習メモ（学習中のコマンド一覧）
└── second.txt         # 練習用ファイル（空ファイル）
```

## ファイル説明

- **first.txt.txt** - Git の練習で学んだコマンドを記録したメモファイル。`git add`、`git commit`、`git rm`、`git status`、`git diff` のコマンドが記載されている。
- **second.txt** - ファイルの追加・削除操作を練習するために作成された空ファイル。

## 使用言語・ツール

- **プログラミング言語**: なし（テキストファイルのみ）
- **使用ツール**: Git
- **記述言語**: 日本語

## 開発ワークフロー

このリポジトリでは以下の Git 基本操作を練習しています：

1. `git add` - ファイルをステージングエリアに追加
2. `git commit` - 変更をコミット
3. `git rm` - ファイルをリポジトリから削除
4. `git status` - 作業ツリーの状態を確認
5. `git diff` - 変更内容の差分を確認

## コミットメッセージの規約

コミットメッセージは**日本語**で記述されています。以下のパターンに従っています：

- 変更内容を簡潔に説明する（例：「git diffを追記」「second.txtの削除」）
- 操作の種類がわかるように記述する（追記、追加、削除など）
